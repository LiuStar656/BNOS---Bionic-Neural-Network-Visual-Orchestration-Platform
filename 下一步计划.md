# 节点与项目导入导出功能实现计划

## 概述

实现节点和项目的打包导出与导入功能，支持 `.bnos`（节点包）和 `.bnosc`（项目包）两种格式。

**核心流程**：
- **导出**：压缩 → 重命名为自定义扩展名 → 保存到目标位置
- **导入**：复制文件 → 重命名为标准压缩格式 → 解压到目标位置

---

## 技术方案

### 文件格式定义

| 格式 | 扩展名 | 内容 | 用途 |
|------|--------|------|------|
| **节点包** | `.bnos` | 单个节点的配置、代码、资源文件 | 节点分享、备份 |
| **项目包** | `.bnosc` | 整个项目的所有节点和配置 | 项目备份、迁移 |

### 压缩格式

- 使用 Python 内置 `zipfile` 模块
- 压缩格式：ZIP_DEFLATED（最大压缩率）
- 密码保护：可选（后续扩展）

### 目录结构

**节点包 `.bnos` 解压后结构**：
```
node_name/
├── config.json      # 节点配置
├── main.py          # 节点代码
├── requirements.txt # 依赖声明（如需要）
└── logs/            # 日志目录（可选）
```

**项目包 `.bnosc` 解压后结构**：
```
project_name/
├── project.json     # 项目配置
├── canvas_layout.json  # 画布布局
└── nodes/           # 节点目录
    ├── node1/
    ├── node2/
    └── ...
```
