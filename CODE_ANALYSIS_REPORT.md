## 3. 已知问题修复记录

### 3.1 已修复的问题

| 问题描述 | 修复位置 | 状态 |
|----------|----------|------|
| 圆形节点锚点和文字重启时坐标错误 | `ui/canvas/canvas_layout.py` | ✅ 已修复 |
| 状态字符串 k 值未映射 | `ui/core/strings_cn.json` | ✅ 已修复 |
| 方块转圆形时线条未刷新 | `ui/canvas/canvas_menus.py` | ✅ 已修复 |

### 3.2 问题修复详情

#### 问题1：圆形节点锚点和文字坐标错误

**原因**：加载布局时，节点样式尺寸未正确恢复，使用默认尺寸计算锚点和文字位置。

**修复**：在 `load_layout` 方法中，三个节点加载路径均先恢复节点尺寸，再更新样式对象尺寸并重新应用样式。

#### 问题2：状态字符串 k 值未映射

**原因**：`resource_monitor.py` 中使用 `t("k_status_running")` 等 k 值，但 `strings_cn.json` 中缺失对应映射。

**修复**：在 `strings_cn.json` 中添加了 `k_status_running`、`k_status_idle`、`k_status_stopped` 的中文映射。

#### 问题3：样式切换时线条未刷新

**原因**：样式切换后，锚点位置变化，但未触发连线路径更新。

**修复**：在 `_switch_node_style` 方法中，收集连接到该节点的所有连线并调用 `edge.update_path()` 刷新路径。
