## 1. 项目概述

### 1.1 背景

BNOS 系统目前存在多个独立实现的菜单系统，功能存在重复和分散，导致开发效率低、维护成本高、用户体验不一致。

### 1.2 问题定义

| 问题 | 影响 | 严重程度 |
|-----|------|---------|
| 功能重复 | 修改功能需改多处 | ⭐⭐⭐ 高 |
| 实现分散 | 代码难以复用 | ⭐⭐⭐ 高 |
| 用户体验不一致 | 相同功能在不同菜单表现不同 | ⭐⭐ 中 |
| 开发效率低 | 新功能开发周期长 | ⭐⭐⭐ 高 |
| 测试难度大 | 每个菜单需独立测试 | ⭐⭐ 中 |

### 1.3 项目目标

| 目标 | 量化指标 | 时间线 |
|-----|---------|--------|
| 代码复用率提升 | 60% 从 ~1500 行到 ~600 行 | 第4周 |
| 开发效率提升 | 50% 新功能开发时间减半 | 第6周 |
| 功能一致性 | 100% 所有菜单功能统一 | 第4周 |
| 维护成本降低 | 40% 减少修改范围 | 长期 |

### 1.4 范围

**包含**：
- ✅ 主菜单栏 (`MenuManager`)
- ✅ 画布右键菜单 (`CanvasMenusMixin`)
- ✅ 节点列表右键菜单 (`NodeListContextMixin`)
- ✅ Action 统一管理和注册

**不包含**：
- ❌ 节点配置对话框（独立组件）
- ❌ 项目文件菜单（已有合理实现）
- ❌ 对话框内部按钮菜单

## 3. 目标设计

### 3.1 目标架构

```
┌─────────────────────────────────────────────────────────────┐
│               目标架构：统一菜单系统                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────────────────────────────────────────────────┐  │
│  │         ActionRegistry (统一功能注册表)              │  │
│  │  • Action Definition (功能定义)                      │  │
│  │  • Action Registry (功能注册)                        │  │
│  │  • Action Factory (Action 创建工厂)                 │  │
│  └──────────────────────────────────────────────────────┘  │
│                          │                                   │
│         ┌────────────────┼────────────────┐                 │
│         ▼                ▼                ▼                 │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐    │
│  │ MenuManager  │  │ CanvasMenus  │  │NodeListMenu  │    │
│  │   (主菜单)   │  │  (画布右键)  │  │ (列表右键)  │    │
│  └──────────────┘  └──────────────┘  └──────────────┘    │
│         (使用统一 ActionFactory，避免重复)                │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 核心设计原则

| 原则 | 说明 |
|-----|------|
| **单一真实来源** | 每个功能只定义一次 |
| **开闭原则** | 对扩展开放，对修改关闭 |
| **依赖倒置** | 依赖抽象（ActionRegistry）而非具体实现 |
| **渐进式迁移** | 避免大爆炸式重构 |
| **向后兼容** | 保持现有接口，逐步迁移 |

### 3.3 用户体验目标

- ✅ **一致性**：相同功能在所有菜单表现相同
- ✅ **可预测性**：用户知道功能在哪里
- ✅ **高效性**：快速找到并执行操作
- ✅ **可发现性**：新用户易于发现功能

from dataclasses import dataclass, field
from typing import Optional, Callable, Any
from enum import Enum

class ActionCategory(Enum):
    """功能分类枚举"""
    NODE = "node"
    CANVAS = "canvas"
    PROJECT = "project"
    VIEW = "view"
    SETTINGS = "settings"
    TOOLS = "tools"

@dataclass
class ActionContext:
    """执行上下文 - 传递给Action的参数"""
    node_name: Optional[str] = None
    node_list: Optional[list[str]] = None
    group_name: Optional[str] = None
    canvas_selection: Optional[Any] = None
    extra: dict = field(default_factory=dict)

@dataclass
class ActionDefinition:
    """功能定义 - 单一真实来源"""
    id: str
    name_i18n: str
    category: ActionCategory
    description_i18n: Optional[str] = None
    shortcut_id: Optional[str] = None
    requires_selection: bool = False
    requires_node: bool = False
    requires_permission: Optional[str] = None
    icon_id: Optional[str] = None
    execute_fn: Optional[Callable[[ActionContext], bool]] = None
    is_enabled_fn: Optional[Callable[[ActionContext], bool]] = None
    is_checked_fn: Optional[Callable[[ActionContext], bool]] = None

#### 4.2.2 ActionRegistry（功能注册表）

```python
# ui/core/actions/action_registry.py
from typing import Dict, Optional, List
from .action_definition import ActionDefinition, ActionContext, ActionCategory

class ActionRegistry:
    """统一功能注册表 - 单例模式"""
    
    _instance: Optional['ActionRegistry'] = None
    _actions: Dict[str, ActionDefinition] = {}
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
    
    @classmethod
    def register(cls, action_def: ActionDefinition):
        """注册功能定义"""
        cls._actions[action_def.id] = action_def
    
    @classmethod
    def get(cls, action_id: str) -> Optional[ActionDefinition]:
        """获取功能定义"""
        return cls._actions.get(action_id)
    
    @classmethod
    def all(cls, category: Optional[ActionCategory] = None) -> List[ActionDefinition]:
        """获取所有功能定义，可按分类过滤"""
        if category is None:
            return list(cls._actions.values())
        return [a for a in cls._actions.values() if a.category == category]
    
    @classmethod
    def execute(cls, action_id: str, context: Optional[ActionContext] = None) -> bool:
        """执行功能"""
        action = cls.get(action_id)
        if not action or not action.execute_fn:
            return False
        
        if context is None:
            context = ActionContext()
        
        if action.requires_permission:
            if not cls._check_permission(action.requires_permission, context):
                return False
        
        return action.execute_fn(context)
    
    @classmethod
    def is_enabled(cls, action_id: str, context: Optional[ActionContext] = None) -> bool:
        """检查功能是否可用"""
        action = cls.get(action_id)
        if not action:
            return False
        if context is None:
            context = ActionContext()
        if action.is_enabled_fn:
            return action.is_enabled_fn(context)
        return True
    
    @classmethod
    def _check_permission(cls, permission: str, context: ActionContext) -> bool:
        """检查权限（预留接口）"""
        return True
```

#### 4.2.3 ActionFactory（Action 创建工厂）

```python
# ui/core/actions/action_factory.py
from PyQt6.QtWidgets import QMenu, QMenuBar
from PyQt6.QtGui import QAction
from typing import Optional
from .action_registry import ActionRegistry
from .action_definition import ActionDefinition, ActionContext
from ui.core.i18n import t
from ui.icons import get_icon

class ActionFactory:
    """QAction 创建工厂"""
    
    @staticmethod
    def create_action(
        parent,
        action_id: str,
        context: Optional[ActionContext] = None,
        menu: Optional[QMenu] = None
    ) -> Optional[QAction]:
        """创建 QAction（从统一注册表）"""
        
        action_def = ActionRegistry.get(action_id)
        if not action_def:
            return None
        
        action = QAction(t(action_def.name_i18n), parent)
        
        if action_def.icon_id:
            action.setIcon(get_icon(action_def.icon_id))
        
        if action_def.shortcut_id and menu is None:
            if hasattr(parent, 'shortcut_mgr'):
                action.setShortcut(parent.shortcut_mgr.get(action_def.shortcut_id))
        
        current_context = context or ActionContext()
        action.setEnabled(ActionRegistry.is_enabled(action_id, current_context))
        
        if action_def.is_checked_fn:
            action.setCheckable(True)
            action.setChecked(action_def.is_checked_fn(current_context))
        
        if action_def.description_i18n:
            action.setStatusTip(t(action_def.description_i18n))
            action.setToolTip(t(action_def.description_i18n))
        
        def on_triggered(checked=False):
            ActionRegistry.execute(action_id, current_context)
        
        action.triggered.connect(on_triggered)
        
        if menu is not None:
            menu.addAction(action)
        
        return action
    
    @staticmethod
    def create_submenu(
        parent,
        menu_title_i18n: str,
        parent_menu: Optional[QMenu] = None,
        menubar: Optional[QMenuBar] = None
    ) -> QMenu:
        """创建子菜单"""
        title = t(menu_title_i18n)
        if menubar:
            return menubar.addMenu(title)
        elif parent_menu:
            return parent_menu.addMenu(title)
        return QMenu(title, parent)
```

# ui/core/actions/builtin_node_actions.py
"""内置节点操作功能定义"""
from .action_definition import ActionDefinition, ActionCategory, ActionContext
from .action_registry import ActionRegistry
from ui.core.i18n import t

def register_node_actions(main_window):
    """注册节点相关功能"""
    
    # ===== 节点启动 =====
    def execute_node_start(ctx: ActionContext) -> bool:
        if ctx.node_name:
            main_window.start_selected_node_by_name(ctx.node_name)
            return True
        elif ctx.node_list:
            for name in ctx.node_list:
                main_window.start_selected_node_by_name(name)
            return True
        main_window.start_selected_node()
        return True
    
    def is_node_start_enabled(ctx: ActionContext) -> bool:
        if ctx.node_name:
            node_info = main_window.nodes_data.get(ctx.node_name, {})
            return node_info.get('status') == 'stopped'
        return True
    
    ActionRegistry.register(ActionDefinition(
        id="node.start",
        name_i18n="k_node_start",
        category=ActionCategory.NODE,
        shortcut_id="start_node",
        requires_node=True,
        description_i18n="_k_start_node_tip",
        execute_fn=execute_node_start,
        is_enabled_fn=is_node_start_enabled,
    ))
    
    # ===== 节点停止 =====
    def execute_node_stop(ctx: ActionContext) -> bool:
        if ctx.node_name:
            main_window.stop_selected_node_by_name(ctx.node_name)
            return True
        main_window.stop_selected_node()
        return True
    
    def is_node_stop_enabled(ctx: ActionContext) -> bool:
        if ctx.node_name:
            node_info = main_window.nodes_data.get(ctx.node_name, {})
            return node_info.get('status') in ('running', 'idle')
        return True
    
    ActionRegistry.register(ActionDefinition(
        id="node.stop",
        name_i18n="k_node_stop",
        category=ActionCategory.NODE,
        shortcut_id="stop_node",
        requires_node=True,
        description_i18n="_k_stop_node_tip",
        execute_fn=execute_node_stop,
        is_enabled_fn=is_node_stop_enabled,
    ))
    
    # ===== 节点配置 =====
    def execute_node_config(ctx: ActionContext) -> bool:
        if ctx.node_name:
            from ui.canvas.canvas_view import CanvasView
            canvas = main_window.get_canvas()
            if canvas:
                canvas.open_node_config(ctx.node_name)
            return True
        return False
    
    ActionRegistry.register(ActionDefinition(
        id="node.config",
        name_i18n="k_node_config",
        category=ActionCategory.NODE,
        requires_node=True,
        execute_fn=execute_node_config,
    ))
    
    # ===== 节点删除 =====
    def execute_node_delete(ctx: ActionContext) -> bool:
        if ctx.node_name:
            if hasattr(main_window, 'node_list_panel'):
                main_window.node_list_panel.delete_node(ctx.node_name)
            return True
        elif ctx.node_list:
            if hasattr(main_window, 'node_list_panel'):
                main_window.node_list_panel._batch_delete_results = {
                    'success': 0, 'failed': 0, 'failed_nodes': []
                }
                # 这里可以调用批量删除逻辑
            return True
        return False
    
    ActionRegistry.register(ActionDefinition(
        id="node.delete",
        name_i18n="k_node_delete",
        category=ActionCategory.NODE,
        requires_node=True,
        execute_fn=execute_node_delete,
    ))
    
    # ===== 导出节点 =====
    def execute_node_export(ctx: ActionContext) -> bool:
        if ctx.node_name:
            main_window.export_node(ctx.node_name)
            return True
        return False
    
    ActionRegistry.register(ActionDefinition(
        id="node.export",
        name_i18n="k_export_node",
        category=ActionCategory.NODE,
        requires_node=True,
        execute_fn=execute_node_export,
    ))

# ui/menu/menu_manager.py (重构后)
"""BNOS 菜单管理器 - 使用统一 ActionRegistry"""
from PyQt6.QtWidgets import QMenuBar
from ui.core.actions.action_factory import ActionFactory
from ui.core.actions.action_registry import ActionRegistry
from ui.core.actions.builtin_project_actions import register_project_actions
from ui.core.actions.builtin_node_actions import register_node_actions
from ui.core.actions.builtin_canvas_actions import register_canvas_actions

class MenuManager:
    """菜单管理器"""
    
    @staticmethod
    def init_menu(main_window, menubar: QMenuBar = None):
        if menubar is None:
            menubar = main_window.menuBar()
        
        # ===== 步骤1: 注册所有内置Action =====
        register_project_actions(main_window)
        register_node_actions(main_window)
        register_canvas_actions(main_window)
        
        # ===== 步骤2: 构建菜单（使用ActionFactory） =====
        
        # --- 文件菜单 ---
        file_menu = ActionFactory.create_submenu(main_window, "k_menu_file", menubar=menubar)
        ActionFactory.create_action(main_window, "project.new", menu=file_menu)
        ActionFactory.create_action(main_window, "project.open", menu=file_menu)

### 附录 B: i18n（k值转化）处理规范

#### B.1 概述

BNOS 系统已实现完整的国际化（i18n）框架，所有显示字符串均使用 `k_` 前缀的 key 通过 `t()` 函数进行翻译。在菜单功能统一化重构中，必须严格遵循此规范。

#### B.2 现有 i18n 架构

```python
# ui/core/i18n.py
from ui.core.i18n import t, init_i18n

init_i18n("cn")  # 初始化中文
msg = t("k_node_start")  # 返回对应翻译字符串
```

**语言文件位置**：
- `ui/core/strings_cn.json` - 中文
- `ui/core/strings_en.json` - 英文

**核心规则**：
1. 所有显示字符串必须使用 `k_` 前缀的 key
2. 通过 `t(key)` 函数获取翻译后的字符串
3. 未找到 key 时返回 key 本身（兜底机制）

#### B.3 重构中的 i18n 处理原则

| 原则 | 说明 |
|-----|------|
| **完全保留 k 值** | ActionDefinition 中只存储 key，不存储翻译后字符串 |
| **延迟翻译** | 在 ActionFactory 创建 QAction 时才调用 t() 翻译 |
| **语言切换支持** | 确保菜单能响应语言切换 |
| **无硬编码字符串** | 所有显示文本都通过 i18n 系统 |

#### B.4 ActionDefinition 中的 i18n 字段

```python
# ui/core/actions/action_definition.py
@dataclass
class ActionDefinition:
    """功能定义 - 完全使用 k 值"""
    id: str
    name_i18n: str              # ✅ 必须是 k_ 前缀的 key，如 "k_node_start"
    category: ActionCategory
    description_i18n: Optional[str] = None  # ✅ 也是 k_ 前缀的 key
    # ... 其他字段
```

**错误示例** ❌：
```python
ActionDefinition(
    id="node.start",
    name_i18n="启动节点",  # ❌ 硬编码中文，不要这样！
    # ...
)
```

**正确示例** ✅：
```python
ActionDefinition(
    id="node.start",
    name_i18n="k_node_start",  # ✅ 使用 k_ 前缀的 key
    description_i18n="_k_start_node_tip",  # ✅ 也是 key
    # ...
)
```

#### B.5 ActionFactory 中的翻译时机

```python
# ui/core/actions/action_factory.py
from ui.core.i18n import t

class ActionFactory:
    @staticmethod
    def create_action(parent, action_id, context=None, menu=None):
        action_def = ActionRegistry.get(action_id)
        if not action_def:
            return None
        
        # ✅ 在创建 QAction 时才翻译（延迟翻译）
        action = QAction(t(action_def.name_i18n), parent)  # 这里翻译
        
        if action_def.description_i18n:
            action.setStatusTip(t(action_def.description_i18n))  # 这里翻译
            action.setToolTip(t(action_def.description_i18n))  # 这里翻译
        
        # ... 其他逻辑
        return action
```

#### B.6 语言切换支持

为了支持运行时切换语言，需要确保：

1. **菜单重建机制**：语言切换时重新构建菜单
2. **Action 更新**：刷新所有 QAction 的文本

```python
# 主窗口中的语言切换处理
def on_language_changed(self, lang):
    set_lang(lang)
    # 重新构建菜单
    MenuManager.init_menu(self, self.menuBar())
    # 刷新其他 UI...
```

#### B.7 新增 k 值规范

如果需要新增功能，需要：

1. **在 strings_cn.json 中添加中文翻译**
2. **在 strings_en.json 中添加英文翻译**
3. **在 ActionDefinition 中使用 k_ 前缀的 key**

**示例**：
```json
// strings_cn.json
{
  "k_node_start": "启动节点",
  "_k_start_node_tip": "启动选中的节点"
}
```

```json
// strings_en.json
{
  "k_node_start": "Start Node",
  "_k_start_node_tip": "Start the selected node"
}
```
