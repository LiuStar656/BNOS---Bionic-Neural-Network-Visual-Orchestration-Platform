# P2 级别优化：主窗口进一步解耦

## 概述

在之前拆分的基础上，进一步将主窗口文件从约1090行精简到 **499行**，通过新增多个 Mixin 模块实现更细粒度的职责分离。本次优化不仅完成了代码结构的重构，还修复了多个潜在问题并增强了代码质量。

## 优化目标

| 目标 | 原始状态 | 优化后 | 达成情况 |
|------|----------|--------|----------|
| 主窗口行数 | ~1500 行 | **499 行** | ✅ 达成 |
| 职责分离 | 单一文件 | 7个独立模块 | ✅ 达成 |
| 代码重复率 | 高 | 低 | ✅ 达成 |
| 跨平台兼容 | Windows 专属 | Windows/macOS/Linux | ✅ 达成 |

### 二、修改文件

#### 1. `ui/main_window.py`

**变更内容**：
- 新增引入 4 个 Mixin 类
- 移除已拆分的方法（面板管理、IPC、节点控制、窗口交互）
- 文件行数从 1090 行减少到 **499行**

**最终继承结构**：
```python
class BNOSMainWindow(QMainWindow, 
                    MainWindowStateMixin, 
                    MainWindowLifecycleMixin, 
                    MainWindowActionsMixin, 
                    MainWindowPanelMixin, 
                    MainWindowIPCMixin, 
                    MainWindowNodeControlMixin, 
                    MainWindowInteractionMixin):
```

#### 2. `ui/main_window_actions.py`

**变更内容**：
- 移除过时的面板创建方法（已移至 `main_window_panel.py`）
- 添加跨平台支持（Windows/macOS/Linux）
- 添加类型注解（提升代码可读性）
- 抽取 `_get_canvas()` 辅助方法消除代码重复
- 增强错误处理（添加参数验证）

**关键改进**：
```python
# 跨平台打开目录
def _open_project_directory(self, project_path: str) -> None:
    if sys.platform == 'win32':
        subprocess.Popen(['explorer', project_path])
    elif sys.platform == 'darwin':
        subprocess.Popen(['open', project_path])
    else:
        subprocess.Popen(['xdg-open', project_path])
```

#### 3. `ui/core/validators.py`

**变更内容**：
- 添加缺失的 `import os`
- 修复路径穿越检查逻辑（使用 `os.path.normpath()` 规范化后再检查）

**关键修复**：
```python
def validate_path(path):
    if not path:
        return False, "路径不能为空"
    
    normalized = os.path.normpath(path)  # 关键修复：规范化路径
    if '..' in normalized.split(os.sep):
        return False, "路径不能包含父目录引用"
    
    return True, ""
```

#### 4. `nodes/*/listener.py`（所有节点）

**变更内容**：
- 修复 Unicode 编码问题（Windows GBK 终端无法处理 emoji 字符）

**关键修复**：
```python
def log(msg, level="INFO"):
    line = f"[{now}] [{level}] {msg}"
    try:
        print(line)
    except UnicodeEncodeError:
        # 替换无法编码的字符
        safe_line = line.encode('gbk', errors='replace').decode('gbk')
        print(safe_line)
```

#### 5. `ui/core/node_process.py`

**变更内容**：
- 添加 `_check_directory_permissions()` 方法进行权限检查
- 提供更友好的错误提示信息
- 在启动节点前先检查目录权限
