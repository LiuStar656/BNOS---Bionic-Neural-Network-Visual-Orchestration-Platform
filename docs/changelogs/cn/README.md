# 更新日志目录

本目录包含 BNOS 项目的所有更新历史记录。

## 目录结构

```
changelogs/
├── cn/                     # 中文更新日志
│   ├── INDEX.md           # 中文更新日志索引
│   ├── README.md          # 本文件
│   ├── 2026-06-07/       # 按日期分类的更新
│   ├── 2026-06-06/
│   └── ...
├── en/                     # 英文更新日志
│   ├── INDEX.md           # 英文更新日志索引
│   ├── README.md
│   └── ...
└── README.md              # 总索引（本文件）
```

## 如何使用

1. 查看所有更新的日期列表：查看 [INDEX.md](./INDEX.md)
2. 查看特定日期的更新：进入对应日期的文件夹
3. 查看单个更新详情：打开对应更新的 `.md` 文件

- 英文版本：[en/](./en/)
