# 01_连线渲染矢量轮廓填充

**日期**: 2026-06-22

## 背景

放大画布后贝塞尔连线与直线段显示明显锯齿阶梯状边缘；QPen 描边在高倍缩放下过渡区过于狭窄；ItemCoordinateCache 缓存位图缩放下产生像素块。

## 变更内容

### 核心思路：矢量轮廓填充

用 `QPainterPathStroker` 将线条的 `QPainterPath` 扩展为闭合轮廓路径，再用 `QBrush` 填充（与箭头三角形多边形填充方式一致）。

```python
# 伪代码：核心思路
stroker_pen = QPen(color, base_w)
stroker_pen.setCapStyle(Qt.PenCapStyle.RoundCap)
stroker_pen.setJoinStyle(Qt.PenJoinStyle.RoundJoin)
stroker = QPainterPathStroker(stroker_pen)
filled_path = stroker.createStroke(self.path())

painter.setPen(QPen(Qt.PenStyle.NoPen))
painter.setBrush(QBrush(color))
painter.drawPath(filled_path)
```

### EdgeItem.paint() 改造

**文件**: `ui/canvas/items/edge_item.py`

```python
def paint(self, painter, option, widget=None):
    painter.setRenderHint(QPainter.RenderHint.Antialiasing, True)
    painter.setRenderHint(QPainter.RenderHint.SmoothPixmapTransform, True)

    base_w = 3.0 * (painter.device().devicePixelRatioF() or 1.0)
    stroker_pen = QPen(color, base_w)
    stroker_pen.setCapStyle(Qt.PenCapStyle.RoundCap)
    stroker_pen.setJoinStyle(Qt.PenJoinStyle.RoundJoin)
    stroker = QPainterPathStroker(stroker_pen)
    filled_path = stroker.createStroke(self.path())
    painter.setPen(QPen(Qt.PenStyle.NoPen))
    painter.setBrush(QBrush(color))
    painter.drawPath(filled_path)
```

**缓存模式**：由 `ItemCoordinateCache` 改为 `NoCache`，避免缓存位图在缩放下锯齿。

### TempEdgeItem（拖拽虚线预览）

同样使用 `QPainterPathStroker` 填充渲染，并设置 `setCacheMode(NoCache)`。

### EdgeArrowItem（箭头三角形）

保持多边形填充，新增 `NoCache` 与 `NoPen` 以确保抗锯齿效果一致。

### 折叠点 / 手柄

折叠点和手柄原本就使用 `painter.drawEllipse()` + `QBrush` 填充，无需改动。

### 设计要点

| 特性 | 实现方式 |
|------|---------|
| **抗锯齿** | `painter.setRenderHint(Antialiasing, True)`，所有连线元素均启用 |
| **矢量填充** | `QPainterPathStroker` + `createStroke(path)` + `setBrush` + `drawPath` |
| **RoundCap / RoundJoin** | `stroker_pen.setCapStyle(RoundCap)` + `setJoinStyle(RoundJoin)` |
| **NoCache** | `EdgeItem` / `TempEdgeItem` / `EdgeArrowItem` 全部改为 `NoCache` |
| **线条宽度** | 基础 `3.0 px`，在 `paint()` 中根据 `devicePixelRatioF()` 动态调整 |

## 修复前后对比

| 场景 | 修复前 | 修复后 |
|------|--------|--------|
| 1x 正常视图 | 细线，无明显问题 | 线条平滑 |
| 2x-3x 放大 | 出现阶梯状锯齿 | 平滑边缘 |
| 5x-10x 放大 | 锯齿非常明显（像素块） | 依旧平滑（矢量填充） |
| 折叠点 / 手柄 | QPainter 填充椭圆，OK | 保持填充方式，不变 |
| 箭头三角形 | 原生多边形填充，OK | 保持填充方式，新增 NoPen |
