# 03_画布分辨率自定义

**日期**: 2026-06-22

## 背景

用户无法自定义画布尺寸与抗锯齿开关。对于特别大的节点工作流需要更大画布，对于简单工作流需要更快渲染。

## 变更内容

### ui/dialogs/settings_dialog.py：新增「渲染」标签页

在设置对话框（`SettingsDialog`）中新增第 4 个标签页「渲染」，包含 5 个预设分辨率按钮 + 自定义宽高输入 + 抗锯齿开关。确定后提示重启生效。

### ui/core/app_config.py：默认配置新增 rendering

```python
DEFAULT_CONFIG = {
    "language": "cn",
    "rendering": {
        "canvas_width": 5000,
        "canvas_height": 5000,
        "antialiasing": True
    },
    # ... 其他字段保持不变
}
```

持久化到 `app_config.json` 后的示例：

```json
{
  "language": "cn",
  "rendering": {
    "canvas_width": 5000,
    "canvas_height": 5000,
    "antialiasing": true
  }
}
```

### ui/canvas/canvas_view.py：读取配置并应用

`NodeCanvas` 初始化时读取 `AppConfig("rendering")` 获取 `canvas_width/canvas_height/antialiasing`，并应用到 `setSceneRect` 和 `setRenderHint`。

### 翻译键

| 键名 | 中文 | English | 用途 |
|------|------|---------|------|
| `settings.rendering.title` | 渲染 | Rendering | 标签页标题 |
| `settings.rendering.preset_1000` | 1000 x 1000 | 1000 x 1000 | 预设按钮 1000 |
| `settings.rendering.preset_2000` | 2000 x 2000 | 2000 x 2000 | 预设按钮 2000 |
| `settings.rendering.preset_5000` | 5000 x 5000 (推荐) | 5000 x 5000 (Recommended) | 预设按钮 5000 |
| `settings.rendering.preset_8000` | 8000 x 8000 | 8000 x 8000 | 预设按钮 8000 |
| `settings.rendering.preset_10000` | 10000 x 10000 | 10000 x 10000 | 预设按钮 10000 |
| `settings.rendering.custom` | 自定义 | Custom | 自定义按钮 |
| `settings.rendering.width` | 宽度 | Width | 宽度标签 |
| `settings.rendering.height` | 高度 | Height | 高度标签 |
| `settings.rendering.px` | 像素 | px | 单位 |
| `settings.rendering.antialiasing` | 启用抗锯齿 | Enable Antialiasing | 抗锯齿开关 |
| `settings.rendering.restart_tip` | 分辨率与抗锯齿配置将在重启后生效。是否立即重启？ | Resolution and antialiasing settings will take effect after restart. Restart now? | 确定后提示 |

### 设计要点

| 特性 | 实现方式 |
|------|---------|
| **验证范围** | 宽度/高度校验：`500 <= value <= 10000`，超出范围回退到默认 5000 |
| **默认值** | 5000 x 5000，推荐预设 |
| **抗锯齿默认** | `true`，保持现有平滑渲染行为 |
| **重启提示** | 配置写入后弹窗询问是否立即重启；选择否则当前会话保持旧画布尺寸 |

## 影响范围

- **修改文件**: `ui/dialogs/settings_dialog.py`、`ui/core/app_config.py`、`ui/canvas/canvas_view.py`、`ui/core/strings_cn.json`、`ui/core/strings_en.json`
- **新增翻译键**: 12 个（`settings.rendering.*` 系列）
- **向后兼容**: 旧版 `app_config.json` 缺少 `rendering` 字段时，使用默认值 5000 x 5000 + 抗锯齿开启
