## 🔧 异步挂载/卸载/刷新节点，不阻塞 GUI (2026-06-05)

### 问题描述

**挂载/卸载/刷新节点时 GUI 无响应**
- **问题**：挂载外部节点、卸载节点、刷新节点列表时，由于操作是同步阻塞的，导致整个 GUI 无响应
- **影响**：用户无法在操作期间进行其他操作，体验不佳

### 修复方案

**异步化以下功能**

1. **外部节点挂载**
   - 立即显示"正在处理挂载请求..."
   - 异步读取配置、注册节点
   - 完成后更新 UI

2. **外部节点卸载**
   - 立即显示"正在卸载节点..."
   - 异步从注册表注销、更新组
   - 完成后更新 UI

3. **刷新节点列表**
   - 立即显示"正在刷新节点列表..."
   - 异步扫描目录、读取配置、恢复挂载节点
   - 完成后更新 UI

### 修改的文件

- `ui/core/external_node_manager.py` - 异步化挂载/卸载
- `ui/core/project_manager.py` - 异步化刷新节点

### 验收标准

✅ 挂载外部节点时 GUI 不阻塞
✅ 卸载外部节点时 GUI 不阻塞
✅ 刷新节点列表时 GUI 不阻塞
✅ 显示过渡状态提示
