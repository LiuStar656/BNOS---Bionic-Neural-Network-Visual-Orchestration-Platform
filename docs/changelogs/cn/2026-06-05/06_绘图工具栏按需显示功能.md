# 🎨 绘图工具栏按需显示功能

## 🎨 绘图工具栏按需显示功能 (2026-06-05)

### 功能改进

**绘图工具栏按需显示（增强版）**
- **问题**：绘图工具栏在画布启动时自动显示并固定在左侧，占用 36px 宽度的画布空间
- **修复**：改为按需显示模式，默认隐藏，用户需要时通过快捷键或菜单切换显示

**新增功能**：
- ✅ **持久化**：工具栏显示状态保存到 `canvas_layout.json`，重启后恢复
- ✅ **右键菜单切换**：在画布右键菜单中添加"显示/隐藏绘图工具栏"选项

### 技术实现

**新增方法**（`draw_layer.py`）：
```python
def show_toolbar(self):
    """显示绘图工具栏"""

def hide_toolbar(self):
    """隐藏绘图工具栏"""

def toggle_toolbar(self):
    """切换绘图工具栏显示/隐藏状态"""
```

**持久化**（`canvas_layout.py`）：
- `save_layout()`：保存 `toolbar_visible` 到 `canvas_layout.json`
- `load_layout()`：加载时恢复工具栏显示状态

**快捷键**：
- `D` 键：切换绘图工具栏显示/隐藏

**右键菜单**：
- "显示绘图工具栏" / "Hide Drawing Toolbar"
- "隐藏绘图工具栏" / "Hide Drawing Toolbar"

**修改的文件**：
- `ui/canvas/draw_layer.py` - 添加工具栏切换方法
- `ui/canvas/canvas_layout.py` - 添加工具栏状态持久化
- `ui/canvas/canvas_menus.py` - 添加右键菜单选项
- `ui/canvas/canvas_view.py` - 添加切换方法
- `ui/core/strings_cn.json` - 添加中文翻译
- `ui/core/strings_en.json` - 添加英文翻译

### 验收标准

✅ 画布启动时绘图工具栏默认隐藏
✅ 按 `D` 键可切换工具栏显示/隐藏
✅ 工具栏隐藏时不影响画布节点操作
✅ 窗口大小变化时隐藏的工具栏不会被意外显示
✅ 工具栏显示状态会持久化到 `canvas_layout.json`
✅ 右键菜单中有"显示/隐藏绘图工具栏"选项
✅ 重启项目后工具栏状态会恢复到上次的状态
