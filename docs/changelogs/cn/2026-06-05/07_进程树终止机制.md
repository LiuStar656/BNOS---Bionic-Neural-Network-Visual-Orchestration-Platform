## 🔨 进程树终止机制 (2026-06-05)

### 功能改进

**彻底终止进程树**
- **问题**：之前的停止节点功能只终止 Python 进程，无法终止其他语言创建的子进程
- **修复**：实现进程树追踪机制，递归查询并终止所有子进程（支持任意语言）

### 技术实现

**进程树查询** (`_get_process_tree`)
- Windows: 使用 WMI 查询 Win32_Process，通过 ParentProcessId 递归查找
- Linux/Mac: 使用 pstree 或 ps 命令查询进程树
- 返回所有进程 PID，按深度优先排序（子进程在前）

**进程树终止** (`_kill_process_tree`)
- 先查询进程树获取所有 PID
- 按顺序终止进程（子进程在前，根进程在后）
- 确保所有子进程都被终止

**停止节点流程**
```
1. 读取 PID 文件获取主进程 PID
2. 调用 _kill_process_tree() 终止进程树
3. 兜底：进程扫描清理残留进程
4. 删除 PID 文件，更新状态
```
