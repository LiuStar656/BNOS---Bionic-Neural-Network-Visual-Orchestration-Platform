# 04 更新日志查看器

## 问题概述

缺少在应用内查看更新日志的功能，用户需要手动浏览 `docs/changelogs/` 文件夹。

## 解决方案

在菜单栏「帮助 → 查看更新日志」中新增入口，根据当前界面语言自动读取中文或英文 changelogs。

## 实现细节

### 1. ChangelogViewer 对话框组件

`ui/core/utils/changelog_viewer.py` — 独立的更新日志查看器

- 自绘深色无边框对话框，黑底白字，与项目整体风格统一
- 通过 `get_lang()` 判断当前语言，自动读取 `docs/changelogs/{cn|en}/README.md`
- 使用 `markdown` 库（`extra` 扩展）将 Markdown + HTML 转换为完整 HTML
- 使用 `QWebEngineView` 渲染，完整支持 `<details>` 点击展开/收起、链接点击等交互

### 2. 菜单与 Action 注册

- 在 `builtin_view_actions.py` 注册 `view.changelog` Action
- 在 `menu_manager.py` 帮助菜单中新增「查看更新日志」菜单项，位于「关于」上方，加分隔线
- 在 `main_window/interaction.py` 新增 `show_changelog()` 方法

### 3. i18n 支持

新增 4 个翻译键：
- `k_menu_changelog` / `k_menu_changelog_desc` — 菜单文本
- `_k_changelog_not_found` / `_k_changelog_read_error` — 错误提示

### 4. 依赖更新

`requirements.txt` 新增 `markdown>=3.5`，用于 Markdown 到 HTML 的转换。

## 验证方式

1. 中文环境下点击「帮助 → 查看更新日志」，应弹出黑底白字对话框，显示中文 changelogs
2. 切换到英文后重复操作，应显示英文 changelogs
3. `<details>` 折叠块应可点击展开/收起
4. 链接可点击，在外部浏览器打开
