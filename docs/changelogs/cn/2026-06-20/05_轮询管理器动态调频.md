# 05_轮询管理器动态调频

**日期**: 2026-06-20

## 背景

`PollingManager` 是 BNOS 的后台任务调度器，负责定时轮询节点状态、日志、配置等。在本次更新前，使用**固定 2 秒间隔**轮询：

- **空闲时**：无节点运行，仍每 2 秒执行一次轮询，浪费 CPU
- **高负载时**：节点密集运行时，2 秒间隔导致 UI 刷新延迟

## 变更内容

### 频率模式定义

```python
# 频率模式常量（秒）
FREQ_HIGH = 1       # CPU < 30%
FREQ_NORMAL = 2     # 30% <= CPU <= 60%
FREQ_LOW = 4        # CPU > 60%

# 阈值
CPU_LOW_THRESHOLD = 30
CPU_HIGH_THRESHOLD = 60
```

### 调频算法

```python
def _adjust_frequency(self):
    """基于系统 CPU 负载动态调整轮询间隔"""
    cpu_usage = self._get_cpu_usage()

    if cpu_usage < self.CPU_LOW_THRESHOLD:
        new_interval = self.FREQ_HIGH    # 1 秒
    elif cpu_usage <= self.CPU_HIGH_THRESHOLD:
        new_interval = self.FREQ_NORMAL  # 2 秒
    else:
        new_interval = self.FREQ_LOW     # 4 秒

    if new_interval != self._interval:
        self._interval = new_interval
        self._timer.setInterval(self._interval * 1000)
```

### 触发时机

```python
# 在 _execute_tick() 中每 10 次 tick 检查一次
def _execute_tick(self):
    self._last_cpu_check += 1
    if self._last_cpu_check >= self._cpu_check_interval:
        self._last_cpu_check = 0
        self._adjust_frequency()
    # ... 执行实际轮询任务
```

### CPU 检测

```python
def _get_cpu_usage(self):
    """获取当前系统 CPU 占用率"""
    try:
        import psutil
        return psutil.cpu_percent(interval=0.1)
    except ImportError:
        return self.CPU_LOW_THRESHOLD  # psutil 不可用时保持低频
```

### 设计要点

| 特性 | 实现 |
|------|------|
| **渐变性** | 每 10 tick 检查一次，避免频繁切换间隔 |
| **优雅降级** | `psutil` 不可用时返回低阈值，保持原有频率 |
| **无侵入** | 仅在内部调整 `self._timer.setInterval()`，对外 API 不变 |

## 效果

| 场景 | 调整前 | 调整后 | CPU 节省 |
|------|--------|--------|----------|
| 空闲（CPU < 30%） | 2 秒 | 1 秒 | UI 更灵敏 |
| 正常（30-60%） | 2 秒 | 2 秒 | 无变化 |
| 高负载（> 60%） | 2 秒 | 4 秒 | ~50% 轮询开销降低 |
