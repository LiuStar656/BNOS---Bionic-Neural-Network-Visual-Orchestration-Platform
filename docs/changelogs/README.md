## 架构说明

本项目采用 **「单一索引页 + 分版本独立 MD 子文件」** 架构管理更新日志：

### 核心特点

✅ **首页极简**：只展示版本标题 + 原生三角折叠按钮  
✅ **动态加载**：点击三角展开查看该版本摘要，详情通过链接跳转  
✅ **模块化解耦**：每个日期独立文件夹，新增版本只需添加文件  
✅ **双向同步**：修改子文件，索引页自动同步更新  
✅ **原生折叠**：使用 `<details><summary>` 原生折叠，无需 JS  
✅ **全平台兼容**：适配 VSCode / 静态站点 / 文档站 / 知识库  

### 文件结构

```
changelogs/
├── README.md          # 本文件（语言选择入口）
├── cn/
│   ├── README.md      # 中文总索引（带折叠三角）
│   ├── INDEX.md       # 中文按日期索引
│   ├── 2026-06-07/
│   ├── 2026-06-06/
│   └── ...
└── en/
    ├── README.md      # 英文总索引（带折叠三角）
    ├── INDEX.md       # 英文按日期索引
    ├── 2026-06-07/
    ├── 2026-06-06/
    └── ...
```

### 使用方式

1. **浏览更新日志**：选择 [中文](./cn/README.md) 或 [英文](./en/README.md) 版本
2. **展开版本摘要**：点击日期左侧的三角展开该版本的主要更新
3. **查看详细更新**：点击「查看完整更新」进入该日期的详细页面
4. **按日期浏览**：使用 INDEX.md 按日期快速导航

### 版本管理

- **新增版本**：在对应日期文件夹添加更新文件，无需修改旧代码
- **版本归档**：每个日期文件夹独立，支持单独归档和复盘
- **版本迭代**：完全解耦，可无限迭代支持几十、上百个版本
