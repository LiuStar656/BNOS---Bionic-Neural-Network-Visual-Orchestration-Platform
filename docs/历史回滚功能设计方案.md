## 1. 功能需求

### 1.1 核心功能

| 功能 | 说明 |
|------|------|
| **撤销（Undo）** | Ctrl+Z，恢复上一步操作 |
| **重做（Redo）** | Ctrl+Y / Ctrl+Shift+Z，重做已撤销的操作 |
| **历史查看** | 显示历史操作列表（可选） |
| **历史清空** | 清空历史记录（打开新项目时） |
| **历史限制** | 最大历史记录数量限制（如 50 步） |

### 1.2 支持的操作类型

| 操作分类 | 具体操作 |
|---------|---------|
| **节点操作** | 创建节点、删除节点、重命名节点、移动节点、修改节点配置 |
| **画布操作** | 创建连线、删除连线、清空画布、移动画布、缩放画布 |
| **视图操作** | 显示/隐藏面板、切换节点样式 |
| **绘图操作** | 绘制内容（Draw Layer） |
| **批量操作** | 批量移动、批量删除等作为一个历史条目 |

### 1.3 非功能需求

| 需求 | 说明 |
|------|------|
| **性能** | 历史记录操作不应影响主线程响应 |
| **稳定性** | 回滚失败时系统应保持一致状态 |
| **持久化** | 历史记录可选保存到项目中（下次打开可继续回滚） |
| **可扩展** | 容易添加新的可撤销操作 |

## 2. 架构设计

### 2.1 设计模式

采用 **Command 模式** + **扁平历史列表**（实现 Photoshop 式历史面板和点击跳转）：

```
┌─────────────────────────────────────────────────────────┐
│                     Main Window                          │
│  ┌───────────────────────────────────────────────────┐ │
│  │          HistoryManager (单例)                     │ │
│  │                                                    │ │
│  │  commands: [C0, C1, C2, C3, C4, C5, C6, C7]      │ │
│  │                          ↑                         │ │
│  │                    current_index = 5               │ │
│  │                                                    │ │
│  │  工作方式：                                         │ │
│  │  - 所有已执行命令按顺序存放在 commands 列表中        │ │
│  │  - current_index 指向当前状态（初始=-1，表示空画布）│ │
│  │  - Undo: current_index-- (回溯一步)                │ │
│  │  - Redo: current_index++ (前进一步)                │ │
│  │  - 跳转: 点历史面板任一项 → 重放到目标 index        │ │
│  │  - 新操作: 如不在末尾，截断 current_index 之后      │ │
│  └───────────────────────────────────────────────────┘ │
│                                                           │
│  ┌───────────────────────────────────────────────────┐ │
│  │  Commands (Command 模式)                          │ │
│  │  - CreateNodeCommand                              │ │
│  │  - DeleteNodeCommand                              │ │
│  │  - MoveNodeCommand                                │ │
│  │  - CreateEdgeCommand                              │ │
│  │  - DeleteEdgeCommand                              │ │
│  │  - MacroCommand (批量包装)                        │ │
│  │  - ...                                            │ │
│  └───────────────────────────────────────────────────┘ │
│                                                           │
│  ┌───────────────────────────────────────────────────┐ │
│  │  HistoryPanel (历史面板 UI)                       │ │
│  │  ┌──────────────────────────────────┐             │ │
│  │  │ 1. 创建节点: data_loader         │             │ │
│  │  │ 2. 移动节点: data_loader         │             │ │
│  │  │ 3. 连接 data_loader → cleaner    │ ← 当前      │ │
│  │  │ 4. 删除节点: cleaner             │ (灰色)      │ │
│  │  │ 5. 创建节点: cleaner             │ (灰色)      │ │
│  │  └──────────────────────────────────┘             │ │
│  │  点击项 3 → 跳转到该状态                         │ │
│  └───────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

### 2.2 与 Photoshop 对照

| 概念 | Photoshop | BNOS |
|------|-----------|------|
| 历史面板 | `Window > History` 面板 | 右侧可停靠 HistoryPanel |
| 历史条目 | 每步操作（含名称/缩略图） | 每步操作（含类型图标/描述/时间） |
| 点击跳转 | 点任一条目跳转 | 点任一条目跳转 |
| 新操作截断 | Undo 后做新操作→ 丢弃灰色条目 | 相同 |
| 快照 | 手动创建命名快照 | MacroCommand 充当批量子集，或独立 Snapshot |
| 最大步数 | 默认 50，可调 | 默认 50，可配置 |

### 2.3 跳转机制：指令重放（轻量级）

```
跳转到 index=3：
  1. 保存 current_index=5
  2. 如果 3 < 5（回退）：从 5 逆序 undo 到 3+1
  3. 如果 3 > 5（前进）：从 5+1 顺序 redo 到 3
  4. current_index = 3
  5. 刷新历史面板高亮
```

**为什么不用快照？** 节点编排场景下，单个 Command 重放极快（创建/移动一个节点是毫秒级操作），50 步内生跳转几乎无感知。相比存储 50 份完整画布快照（每份数 MB），指令重放内存友好得多。

### 2.4 模块依赖关系

```
HistoryManager
    ├── 依赖: ActionSystem (现有)
    ├── 依赖: CanvasLayoutMixin (现有)
    ├── 依赖: EventBus (用于历史变更通知)
    └── 使用: Command 接口

HistoryPanel (UI)
    ├── 依赖: HistoryManager
    ├── 继承: QDockWidget / QWidget
    └── 连接: HistoryManager.history_changed → 刷新列表

## 3. 数据结构

### 3.1 Command 基类

```python
# 文件: ui/core/commands/base.py

from abc import ABC, abstractmethod
from dataclasses import dataclass
from typing import Any, Optional
from enum import Enum


class CommandType(Enum):
    """命令类型"""
    NODE_CREATE = "node.create"
    NODE_DELETE = "node.delete"
    NODE_MOVE = "node.move"
    NODE_RENAME = "node.rename"
    NODE_CONFIG = "node.config"
    EDGE_CREATE = "edge.create"
    EDGE_DELETE = "edge.delete"
    CANVAS_CLEAR = "canvas.clear"
    CANVAS_VIEW = "canvas.view"
    DRAWING = "drawing"
    BATCH = "batch"
    COMPOUND = "compound"


@dataclass
class CommandResult:
    """命令执行结果"""
    success: bool
    message: str = ""
    error: Optional[Exception] = None


class Command(ABC):
    """命令基类"""
    
    def __init__(self, command_type: CommandType, description: str = ""):
        self.command_type = command_type
        self.description = description
        self.timestamp = None  # 执行时间戳
        self.executed = False  # 是否已执行
        self.can_undo = True  # 是否可撤销
    
    @abstractmethod
    def execute(self) -> CommandResult:
        """执行命令"""
        pass
    
    @abstractmethod
    def undo(self) -> CommandResult:
        """撤销命令"""
        pass
    
    @abstractmethod
    def redo(self) -> CommandResult:
        """重做命令（默认调用 execute）"""
        pass
    
    def to_dict(self) -> dict:
        """序列化为字典（用于持久化）"""
        return {
            "type": self.command_type.value,
            "description": self.description,
            "timestamp": self.timestamp,
            "can_undo": self.can_undo,
            "data": self._serialize_data()
        }
    
    @classmethod
    def from_dict(cls, data: dict) -> 'Command':
        """从字典反序列化"""
        pass
    
    @abstractmethod
    def _serialize_data(self) -> dict:
        """序列化命令数据"""
        pass
```

### 3.2 HistoryState（扁平列表 + 指针）

```python
# 文件: ui/core/commands/history_manager.py

from dataclasses import dataclass, field
from typing import List, Optional


@dataclass
class HistoryState:
    """历史记录状态（扁平列表 + 当前指针，实现 Photoshop 式跳转）

    核心数据结构：
      commands:     [C0, C1, C2, C3, C4]  ← 所有已执行命令（按时间序）
      current_index: 3                    ← 指向 C3（当前状态）

    关键语义：
      - current_index = -1 表示初始状态（空画布，未执行任何命令）
      - commands[0] ~ commands[current_index] 构成"已生效"区间
      - commands[current_index+1:] 是"可重做"区间
      - 在非末尾位置执行新命令 → 截断 current_index 之后的所有命令
    """
    commands: List[Command] = field(default_factory=list)
    current_index: int = -1
    max_history: int = 50
    is_recording: bool = True

    def get_can_undo(self) -> bool:
        return self.current_index >= 0

    def get_can_redo(self) -> bool:
        return self.current_index < len(self.commands) - 1

    def get_current_command(self) -> Optional[Command]:
        """获取当前状态的命令"""
        if 0 <= self.current_index < len(self.commands):
            return self.commands[self.current_index]
        return None

    def get_undo_description(self) -> Optional[str]:
        """获取下一步撤销的描述"""
        cmd = self.get_current_command()
        return cmd.description if cmd else None

    def get_redo_description(self) -> Optional[str]:
        """获取下一步重做的描述"""
        redo_index = self.current_index + 1
        if redo_index < len(self.commands):
            return self.commands[redo_index].description
        return None

    def get_all_descriptions(self) -> List[dict]:
        """获取全部历史条目信息（供 HistoryPanel 显示）

        Returns:
            [{index, description, command_type, is_current, is_future}, ...]
        """
        result = []
        for i, cmd in enumerate(self.commands):
            result.append({
                "index": i,
                "description": cmd.description,
                "command_type": cmd.command_type.value,
                "is_current": (i == self.current_index),
                "is_future": (i > self.current_index),
            })
        return result

    def truncate_future(self):
        """截断当前指针之后的所有命令（在新操作执行前调用）"""
        if self.current_index + 1 < len(self.commands):
            self.commands = self.commands[:self.current_index + 1]

    def trim_head(self):
        """超出最大历史限制时，从头部丢弃最旧命令"""
        while len(self.commands) > self.max_history:
            self.commands.pop(0)
            self.current_index -= 1
```

## 4. 核心模块设计

### 4.1 HistoryManager（历史管理器 — 扁平列表 + 跳转）

```python
# 文件: ui/core/commands/history_manager.py

import os
import json
import time
from typing import Optional, List
from PySide6.QtCore import QObject, Signal
from ui.core.logger import logger
from ui.core.event_bus import event_bus


class HistoryManager(QObject):
    """历史记录管理器 - 单例模式（扁平列表 + 指针，支持 Photoshop 式跳转）"""

    # 信号
    history_changed = Signal()       # 历史内容变更（HistoryPanel 刷新）
    can_undo_changed = Signal(bool)  # 是否可撤销状态变更
    can_redo_changed = Signal(bool)  # 是否可重做状态变更
    index_changed = Signal(int)      # 当前指针变更（HistoryPanel 高亮切换）

    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
            cls._instance._initialized = False
        return cls._instance

    def __init__(self):
        if self._initialized:
            return

        super().__init__()
        self.state = HistoryState()
        self._initialized = True

        # 注册到 EventBus
        event_bus.subscribe('project.opened', self._on_project_opened)
        event_bus.subscribe('project.closed', self._on_project_closed)

    def execute_command(self, command: Command) -> CommandResult:
        """执行命令并记录到历史列表

        Photoshop 行为：
          - 如果在历史中间（current_index < len-1），先截断后面的"未来"命令
          - 然后 append 新命令，current_index 指向末尾
        """
        if not self.state.is_recording:
            return command.execute()

        # 执行命令
        result = command.execute()
        if not result.success:
            return result

        # 如果在历史中间（非末尾），截断后面的"未来"
        if self.state.current_index < len(self.state.commands) - 1:
            self.state.truncate_future()

        # 记录时间戳
        command.timestamp = time.time()
        command.executed = True

        # 追加到列表，指针指向末尾
        self.state.commands.append(command)
        self.state.current_index = len(self.state.commands) - 1

        # 限制最大历史记录数
        self.state.trim_head()

        # 发送变更信号
        self._emit_change_signals()
        self.index_changed.emit(self.state.current_index)

        logger.debug("Command: %s (index=%d)", command.description, self.state.current_index)
        return result

    def undo(self) -> CommandResult:
        """撤销一步"""
        if not self.state.get_can_undo():
            return CommandResult(False, "没有可撤销的操作")

        cmd = self.state.get_current_command()
        result = cmd.undo()

        if result.success:
            self.state.current_index -= 1
            self._emit_change_signals()
            self.index_changed.emit(self.state.current_index)
            logger.debug("Undo: %s → index=%d", cmd.description, self.state.current_index)

        return result

    def redo(self) -> CommandResult:
        """重做一步"""
        if not self.state.get_can_redo():
            return CommandResult(False, "没有可重做的操作")

        next_index = self.state.current_index + 1
        cmd = self.state.commands[next_index]
        result = cmd.redo()

        if result.success:
            self.state.current_index = next_index
            self._emit_change_signals()
            self.index_changed.emit(self.state.current_index)
            logger.debug("Redo: %s → index=%d", cmd.description, self.state.current_index)

        return result

    def jump_to(self, target_index: int) -> CommandResult:
        """跳转到指定的历史位置（Photoshop 核心交互）

        通过正向/逆向重放指令到达目标状态。
        复杂度 O(|current - target|)，50 步内生跳转 < 200ms。

        Args:
            target_index: 目标位置。 -1 = 初始空状态， 0~len-1 = 命令列表索引
        """
        current = self.state.current_index

        if target_index == current:
            return CommandResult(True, "已在目标状态")

        if target_index < -1 or target_index >= len(self.state.commands):
            return CommandResult(False, f"无效的目标索引: {target_index}")

        try:
            if target_index < current:
                # 回退：逆向 undo 到目标位置
                for i in range(current, target_index, -1):
                    result = self.state.commands[i].undo()
                    if not result.success:
                        # 回退失败 → 尝试回滚到失败点之前的稳定状态
                        logger.error("跳转失败: undo index=%d 出错", i)
                        return CommandResult(False, f"撤销 {self.state.commands[i].description} 失败")
            else:
                # 前进：正向 redo 到目标位置
                for i in range(current + 1, target_index + 1):
                    result = self.state.commands[i].redo()
                    if not result.success:
                        logger.error("跳转失败: redo index=%d 出错", i)
                        return CommandResult(False, f"重做 {self.state.commands[i].description} 失败")

            self.state.current_index = target_index
            self._emit_change_signals()
            self.index_changed.emit(target_index)
            logger.info("跳转到 index=%d (移动了 %d 步)", target_index, abs(target_index - current))
            return CommandResult(True)

        except Exception as e:
            logger.error("跳转异常: %s", e)
            return CommandResult(False, str(e))

    def clear_history(self):
        """清空历史记录（重置到初始状态）"""
        self.state.commands.clear()
        self.state.current_index = -1
        self._emit_change_signals()
        self.index_changed.emit(-1)
        logger.info("History cleared")

    # ── 历史查看接口 ──

    def get_history_entries(self) -> List[dict]:
        """获取全部历史条目（供 HistoryPanel 渲染）

        Returns:
            [{index, description, command_type, is_current, is_future}, ...]
        """
        return self.state.get_all_descriptions()

    def get_current_index(self) -> int:
        return self.state.current_index

    def can_undo(self) -> bool:
        return self.state.get_can_undo()

    def can_redo(self) -> bool:
        return self.state.get_can_redo()

    # ── 录制控制 ──

    def begin_macro(self, description: str) -> 'MacroCommand':
        """开始宏（批量操作包装为一个历史条目）"""
        from .compound_commands import MacroCommand
        return MacroCommand(description)

    def pause_recording(self):
        self.state.is_recording = False

    def resume_recording(self):
        self.state.is_recording = True

    # ── 持久化（可选）──

    def save_history(self, project_path: str):
        """保存历史到项目文件"""
        if not project_path:
            return

        history_file = os.path.join(project_path, "history.json")
        data = {
            "commands": [cmd.to_dict() for cmd in self.state.commands],
            "current_index": self.state.current_index,
        }

        try:
            with open(history_file, "w", encoding="utf-8") as f:
                json.dump(data, f, indent=2, ensure_ascii=False)
            logger.debug("History saved to %s", history_file)
        except Exception as e:
            logger.error("Failed to save history: %s", e)

    def load_history(self, project_path: str):
        """从项目文件加载历史"""
        # TODO: 实现历史加载
        pass

    def _emit_change_signals(self):
        self.history_changed.emit()
        self.can_undo_changed.emit(self.can_undo())
        self.can_redo_changed.emit(self.can_redo())

    def _on_project_opened(self, project_path: str):
        self.clear_history()

    def _on_project_closed(self):
        self.clear_history()
```

```python
# 文件: ui/core/commands/compound_commands.py

from .base import Command, CommandType, CommandResult
from typing import List


class MacroCommand(Command):
    """宏命令 - 组合多个命令作为一个历史条目"""
    
    def __init__(self, description: str = ""):
        super().__init__(CommandType.COMPOUND, description or "批量操作")
        self._commands: List[Command] = []
    
    def add_command(self, command: Command):
        """添加子命令"""
        self._commands.append(command)
    
    def execute(self) -> CommandResult:
        for cmd in self._commands:
            result = cmd.execute()
            if not result.success:
                return result
        
        if not self.description:
            self.description = f"{len(self._commands)} 个操作"
        
        return CommandResult(True)
    
    def undo(self) -> CommandResult:
        # 逆序撤销
        for cmd in reversed(self._commands):
            result = cmd.undo()
            if not result.success:
                return result
        
        return CommandResult(True)
    
    def redo(self) -> CommandResult:
        return self.execute()
    
    def _serialize_data(self) -> dict:
        return {
            "commands": [cmd.to_dict() for cmd in self._commands]
        }
```

# 文件: ui/panels/history_panel.py

from PySide6.QtWidgets import (QDockWidget, QWidget, QVBoxLayout, QListWidget,
                              QListWidgetItem, QPushButton, QHBoxLayout, QLabel)
from PySide6.QtCore import Qt, Signal
from PySide6.QtGui import QIcon, QFont
from ui.core.commands.history_manager import history_manager
from ui.core.logger import logger


class HistoryPanel(QDockWidget):
    """历史面板（Photoshop 风格）

    功能：
      - 显示所有历史操作列表
      - 当前状态高亮（蓝色/粗体）
      - "未来"状态灰色显示
      - 点击任意条目跳转到该状态
      - 清空历史按钮
    """

    jump_requested = Signal(int)  # 请求跳转到 index

    def __init__(self, parent=None):
        super().__init__("历史记录", parent)
        self.setObjectName("HistoryPanel")
        self.setAllowedAreas(Qt.DockWidgetArea.RightDockWidgetArea |
                             Qt.DockWidgetArea.LeftDockWidgetArea)

        # 主容器
        container = QWidget()
        layout = QVBoxLayout(container)
        layout.setContentsMargins(4, 4, 4, 4)

        # 标题栏
        header_layout = QHBoxLayout()
        self._title_label = QLabel("历史记录")
        self._title_label.setFont(QFont("", 10, QFont.Weight.Bold))
        header_layout.addWidget(self._title_label)
        header_layout.addStretch()

        self._clear_btn = QPushButton("清空")
        self._clear_btn.setFixedSize(50, 22)
        self._clear_btn.clicked.connect(self._on_clear)
        header_layout.addWidget(self._clear_btn)
        layout.addLayout(header_layout)

        # 历史列表
        self._list = QListWidget()
        self._list.setAlternatingRowColors(True)
        self._list.setSelectionMode(QListWidget.SelectionMode.SingleSelection)
        self._list.itemClicked.connect(self._on_item_clicked)
        layout.addWidget(self._list)

        # 底部信息栏
        self._info_label = QLabel("")
        self._info_label.setStyleSheet("color: gray; font-size: 10px;")
        layout.addWidget(self._info_label)

        self.setWidget(container)

        # 连接 HistoryManager 信号
        history_manager.history_changed.connect(self._refresh)
        history_manager.index_changed.connect(self._highlight_index)

        self._refresh()

    # ── 刷新 ──

    def _refresh(self):
        """从 HistoryManager 拉取全量数据并刷新列表"""
        self._list.clear()
        entries = history_manager.get_history_entries()
        current_idx = history_manager.get_current_index()

        for entry in entries:
            item = self._make_item(entry)
            self._list.addItem(item)

            # 高亮当前项
            if entry["is_current"]:
                self._list.setCurrentRow(entry["index"])
                self._highlight_item(item, current=True)

        # 滚动到底部（最新操作）
        if self._list.count() > 0:
            self._list.scrollToBottom()

        # 更新底部信息
        total = len(entries)
        current = current_idx + 1 if current_idx >= 0 else 0
        self._info_label.setText(f"第 {current}/{total} 步"
                                 f"（共 {total} 条记录）")

    def _highlight_index(self, index: int):
        """切换高亮到指定索引（不完全刷新列表）"""
        if 0 <= index < self._list.count():
            # 取消旧高亮
            old_row = self._list.currentRow()
            if old_row >= 0 and old_row != index:
                old_item = self._list.item(old_row)
                if old_item:
                    self._highlight_item(old_item, current=False)

            self._list.setCurrentRow(index)
            new_item = self._list.item(index)
            if new_item:
                self._highlight_item(new_item, current=True)

    def _make_item(self, entry: dict) -> QListWidgetItem:
        """构造列表项"""
        desc = entry["description"]
        if entry["is_future"]:
            desc = f"[未来] {desc}"

        item = QListWidgetItem(desc)
        item.setData(Qt.ItemDataRole.UserRole, entry["index"])

        if entry["is_future"]:
            self._highlight_item(item, future=True)

        return item

    def _highlight_item(self, item: QListWidgetItem, *,
                        current: bool = False, future: bool = False):
        """设置列表项的视觉样式"""
        font = item.font()
        if current:
            font.setBold(True)
            item.setFont(font)
            item.setForeground(Qt.GlobalColor.white)
            item.setBackground(Qt.GlobalColor.darkCyan)
        elif future:
            font.setItalic(True)
            item.setFont(font)
            item.setForeground(Qt.GlobalColor.gray)

    # ── 交互 ──

    def _on_item_clicked(self, item: QListWidgetItem):
        """点击历史条目 → 跳转到对应状态"""
        target_index = item.data(Qt.ItemDataRole.UserRole)
        if target_index is None:
            return

        logger.debug("HistoryPanel: 点击跳转到 index=%d", target_index)
        result = history_manager.jump_to(target_index)

        if not result.success:
            # 跳转失败，恢复列表高亮
            self._highlight_index(history_manager.get_current_index())
            logger.warning("跳转失败: %s", result.message)

    def _on_clear(self):
        """清空历史"""
        history_manager.clear_history()
        self._refresh()
```

## 8. 边界情况处理

| 边界情况 | 处理方案 |
|---------|---------|
| 同时操作多个节点 | 使用 MacroCommand 包装为一个历史条目 |
| 操作失败时 | 不添加到历史记录 |
| 历史记录数量限制 | FIFO 策略，超过时删除最早的记录 |
| 保存/加载历史 | 可选功能，默认不保存（避免文件过大） |
| 项目切换 | 清空当前历史记录 |
| 不可撤销的操作 | 设置 command.can_undo = False |
